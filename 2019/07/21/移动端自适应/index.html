在react项目搭建完成并能成功运行的情况下

step1

安装lib-flexible，postcss-px2rem

npm i lib-flexible postcss-px2rem –save

step2

引入lib-flexible

在项目入口文件index.js文件中引入lib-flexible包

import ‘lib-flexible’

step3

在config/webpack.config.js文件里,进行修改

const px2rem = require(‘postcss-px2rem’)

plugins下:

px2rem({ remUnit: 75 }) //px2rem({ remUnit: 75 })的意思就是1rem = 75px这个是根据750px设计稿来的，如果是620的就写62
